# VR Physics Framework
VRPF is a package that uses physics to give the best interaction possible. Remember that this is a FREE BETA package so dont expect it to be the best. it'll feel more and more like the marrow framework over time.


# VRPF Features

1. Physics based rig.
2. Non physical rig, uses physics based hands nothing more. Simple and nice for small projects.
3. Custom Grabbing
4. Custom Climbing
5. Custom Rigidbodies
6. Physics Based Walking
7. Physics Based Turning
8. Classic Hands
9. Custom Hands
10. 6+ Interaction Helpers
11. UnityEngine Scripts
12. 2+ Extensions
13. Jumping
14. Guns (AK-47, pistol)
15. Grab Posing
16. Two Hand Posing Support
17. Dynamic Posing
18. Full Body IK
19. Auto Set Full Body IK
